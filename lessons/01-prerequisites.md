# Lesson 01: Prerequisites

In this tutorial, we will be developing a simple bubble popping game with pygame-ce and pymunk. Before we begin, we need to install the tools we will be using for this project:
01. install Visual Studio Code from https://code.visualstudio.com/download
02. start Visual Studio Code
03. click the Extensions tab
04. search for "Python"
05. install the Python extension by Microsoft
06. after the extension is installed, close Visual Studio Code
07. install Python
    * Windows: install Python from https://www.python.org/downloads/
    * Linux: install/update Python via your package manager
    * Mac: install/update Python via your package manager
08. install python-venv
    * Windows: n/a
    * Linux: install/update python-venv via your package manager
    * Mac: install/update python-venv via your package manager
